## Getting Started

First, run the development server:

```bash
npm run dev -- -p 5173
#or
npm run dev -- -p 3333
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173] [http://localhost:3333](http://localhost:5173 or http://localhost:3333) with your browser to see the result.
